#### Como colaborar?

A idéia do projeto é ser aberto e todos poderem colaborar com evoluções, correções, melhores documentações, fotos e desenhos.

Segue abaixo um passo a passo simplificado de como colaborar com o projeto

 - Voce precisa se cadastrar no GitHub, oque não vou explicar, porque afinal é só um cadastro!

- Acessar a página principal do repósitorio, neste exemplo vamos corrigir alguns acentos na descrição principal do projeto, e clicar no arquivo **README.md**

  <img  height="450" src="https://raw.githubusercontent.com/DIYGrow/indoor-growbox/master/como-colaborar/imagens/passo1.png"/>

- Feito isso você irá ser levado para a tela de detalhes deste arquivo, para abrir o modo de edição, clique no Lápis destacado na imagem

  <img  height="450" src="https://raw.githubusercontent.com/DIYGrow/indoor-growbox/master/como-colaborar/imagens/passo2.png"/>

- Agora voce pode editar o arquivo como preferir, e também verificar como esta ficando sua alteraçao clicando na aba 'Preview Changes'
  <img  height="300" src="https://raw.githubusercontent.com/DIYGrow/indoor-growbox/master/como-colaborar/imagens/passo3.png"/> <img  height="300" src="https://raw.githubusercontent.com/DIYGrow/indoor-growbox/master/como-colaborar/imagens/passo3_2.png"/>

- Depois de realizar as edições que você quiser, role até o final da página e voce vai encotrar este formulário, voce precisa inserir um titulo para as alterações realizadas e sempre que possível uma descrição das mesmas, em seguida clique no botão 'Propose file change'

<img  height="450" src="https://raw.githubusercontent.com/DIYGrow/indoor-growbox/master/como-colaborar/imagens/passo4.png"/>

- Após clicar em 'Propose file change' você será levado para uma tela que permite que voce revise suas alterações, depois de revisar, clique no botão 'Create pull request'

<img  height="450" src="https://raw.githubusercontent.com/DIYGrow/indoor-growbox/master/como-colaborar/imagens/passo5.png"/>

- Voce precisa inserir um titulo para as alterações realizadas e sempre que possível uma descrição das mesmas, em seguida clique no botão 'Create pull request'

<img  height="450" src="https://raw.githubusercontent.com/DIYGrow/indoor-growbox/master/como-colaborar/imagens/passo6.png"/>

- Feito isso, pronto, voce realizou as alterações que desejava no projeto e enviou uma solicitação para o dono do projeto aceitar suas alterações! Oficialmente você colaborou com um projeto Open Source =D

<img  height="450" src="https://raw.githubusercontent.com/DIYGrow/indoor-growbox/master/como-colaborar/imagens/passo7.png"/>

- Assim que sua alteração for 'aceita' ou 'recusada' ou forem solicitadas alterações nas coisas que voce mudou, você será informado por Email!

- Todos as pessoas que contribuirem aparecem como [Contribuidores do projeto](https://github.com/DIYGrow/indoor-growbox/graphs/contributors "Contribuidores do projeto")

<img  height="450" src="https://raw.githubusercontent.com/DIYGrow/indoor-growbox/master/como-colaborar/imagens/passo8.png"/>
